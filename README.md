## Exécution de l'Algorithme
Ce guide explique comment lancer le programme AlgoBox contenu dans le dépôt.

##Prérequis
*Avoir le logiciel AlgoBox sur votre ordinateur.
*Avoir cloné le dépôt GitHub sur votre ordinateur.

##Etapes pour ouvrir et Exécuter
1. Mise à jour
  * Avant de commencer, ouvrir Git Bash.
  * Placez vosu dans le dossier du projet (`cd Culture_numerique`).
  * Récupérer les dernières modifications du projet (`git pull origin main`)

2. Localiser le fichier
   * Ouvrez l'explorateur de fichier de votre ordinateur et trouvez le dossier du projet.
   * Identifier le fichier principam de l'algorithme : `Programmes.algo`.

3. Lancer AlgoBox
   * Double-cliquer sur le fichier `Programme.algo`.
   * Le programme s'ouvre directement dans AlgoBox.
  
4. Exécuter
   * Sur AlgoBox, cliquer sur "Tester Algorithme" ou sur le bouton "Tester".
   * La fenêtre AlgoBox Test, puis cliquer sur "Lancer Algorithme".
   * Les résultats apparaissent sur la console.
